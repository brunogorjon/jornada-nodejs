🚀 Lógica e Fundamentos: Node.js & JavaScript
Este diretório contém os materiais e exercícios práticos desenvolvidos durante o Intensivo de Fundamentos, focado na transição para o ecossistema Node.js e no aprimoramento da lógica de programação.

📅 Sobre o Curso
Período: Março de 2026

Carga Horária: 8 horas de aulas semanais (Total de 32 horas/mês)

Objetivo: Revisão profunda de lógica de programação utilizando JavaScript e introdução ao ambiente de execução Node.js, culminando na criação dos primeiros elementos de uma API.

📂 Estrutura das Pastas
O conteúdo está organizado por dias de aula, facilitando o acompanhamento da evolução técnica. Cada diretório contém:

Explicação/Exemplos: Códigos desenvolvidos durante a explicação do professor.

Exercícios de Fixação: Desafios individuais propostos para consolidar o aprendizado do dia.

🛠️ O que foi aprendido?
Lógica e JS Core
Trabalho com tipos primitivos e estruturas de dados complexas.

Domínio de laços de repetição e condicionais.

Manipulação avançada de listas (map, filter, reduce).

Node.js Inicial
Entendimento do Event Loop e processamento assíncrono.

Uso de módulos nativos e CommonJS/ES Modules.

Construção de API: Criação de rotas básicas, tratamento de requisições e respostas (HTTP Methods).
